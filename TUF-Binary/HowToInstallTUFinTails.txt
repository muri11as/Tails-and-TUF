HOW TO INSTALL TUF BINARY IN TAILS

Requirements:
- TUF current version is 0.7.5
- Tails current version is 0.21
I. MANUAL
Steps:
1. Download and extract all packages in this repository diretory to /home/amnesia/Desktop
2. Install dependencies
  i. Install pip
     cd pip-1.4.1
     sudo python setup.py install
  ii. Install setuptools
     cd setuptools-1.4
     sudo python setup.py install
  iii. Install wheel
     cd wheel-0.22.0
     sudo python setup.py install
3. Install TUF
   sudo pip install --use-wheel --no-index --find-links="/home/amnesia/Desktop/" tuf
      
II. AUTOMATED SCRIPT
Steps:
1. Download and extract tuf-binary-installation.tar.gz to a folder ie. /home/amnesia/Desktop/
2. Run sudo sh setup.sh
